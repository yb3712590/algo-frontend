# Algo Visual Lab

This repository is an AI frontend test project.

## Project Statement

- This project is created as an experimental AI-assisted frontend implementation.
- The site focuses on algorithm learning and visualization.

## Content Attribution

- Outline and chapter structure are based on **Introduction to Algorithms (CLRS)**.
- Part of the chapter explanation content is summarized from:
  - `datawhalechina/leetcode-notes`
  - Repository: https://github.com/datawhalechina/leetcode-notes

## Tech Stack

- React + Vite + TypeScript
- Tailwind CSS
- Framer Motion

## Publish

- Requirement: install system-level `gh` and complete `gh auth login`.
- Auto create GitHub remote repo and push:
  - `npm run publish:github`
- Create private repo and push:
  - `npm run publish:github:private`
